# Phishing Email Detector

A Python-based phishing email detection tool with a Tkinter GUI.

## Features
- Detects suspicious phishing keywords
- Risk scoring system
- Color-coded risk levels
- Email file upload support
- Risk score graph visualization
- Pie chart dashboard
- Analysis history tracking
- CSV export for reporting
- Timestamped analysis records

## Technologies Used
- Python
- Tkinter
- Matplotlib
- Pandas
